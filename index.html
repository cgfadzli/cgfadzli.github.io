BUILD A WEBSITE (15%)

 

 

Work in group, form a company that provide a professional service of your choice (example: fotography service, business management, event management, wedding management, marketing, fashion, etc.). Give your company a name and a logo. Build a HTML website to promote your company. Design your company website with a proper CSS style. Your company website must have:

 

i. A homepage/landing page that explain about the services offered. (2%)

ii. A contact page that provide a form for user to fill in if they want to contact you. (3%)

iii. A portfolio page that showcase your previous projects to attracts clients. (5%)

iv. A team page that list all the team members with detail information (5%)

 

HOST AND MANAGE YOUR WEBSITE USING GITHUB (5%)

 

Every team member must sign up for a personal github account at https://github.com. Team leader must create a repository for the company website. Team leader must invite all team member as collaborator. You must collaboratively build and host your website using github public repository with https://xxx.github.io URL (replace xxx with your github name). You can learn how to build a website in github through this video:

 

https://www.youtube.com/watch?v=8hrJ4oN1u_8

 

 

PREPARE A VIDEO PRESENTATION

 

Prepare a 2 minutes (maximum) video presentation to show and explain the website that you had build. Upload your video in youtube.

 

Your project will be marked according to the video presentation and link given.
